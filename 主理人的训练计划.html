居家减脂塑形周计划

无器械 · 精准训练 · 健康饮食
周一 全身塑形
🏃 运动计划
热身激活 (5-8分钟)
- 关节活动：转动脚踝、膝盖、髋部、手腕、肩膀、颈部（各30秒）
- 开合跳：40秒
- 高抬腿：40秒
全身力量 (25分钟)
- 深蹲：3组，每组12-15次（组间休息45秒）
- 跪姿俯卧撑：3组，每组做到力竭（组间休息60秒）
- 交替弓步蹲：3组，每条腿10-12次（组间休息45秒）
- 平板支撑：3组，每组坚持30-45秒（组间休息60秒）
- 仰卧臀桥：3组，每组15次（组间休息30秒）
拉伸放松 (5分钟)
- 重点拉伸大腿、臀部、胸部、手臂
B站跟练视频推荐
🍽️ 三餐搭配
早餐 (7:00-8:00)
水煮蛋1个 + 全麦面包1片 + 无糖牛奶/豆浆200ml
午餐 (12:00-13:00)
糙米饭（一拳头大小） + 清蒸鸡胸肉/鱼肉（一掌心大小） + 清炒蔬菜（两大捧）
晚餐 (18:00-19:00)
蒸红薯（一小个） + 豆腐菌菇汤（一大碗） + 凉拌菠菜
核心提示
力量日是塑形基础，感受肌肉发力。动作质量大于数量。
周二 有氧燃脂
🏃 运动计划
热身 (5分钟)
- 关节活动和动态拉伸
持续有氧 (30分钟)
- 选择一种：低冲击有氧操 或 原地慢跑+高抬腿/后踢腿组合
- 保持微喘但能说话的程度
拉伸 (5分钟)
- 重点拉伸大腿后侧、小腿、腰部
B站跟练视频推荐
🍽️ 三餐搭配
早餐
燕麦粥（40g燕麦） + 鸡蛋1个 + 苹果/橙子半个
午餐
杂粮饭（一拳头大小） + 烤瘦肉（一掌心） + 大份蔬菜沙拉（油醋汁）
晚餐
杂粮粥（一碗） + 清炒虾仁（一手掌） + 焯水西兰花
核心提示
保持心率，持续燃烧脂肪。有氧运动是减脂的关键。
周三 积极恢复
🏃 运动计划
主动恢复日 (可选，15-20分钟)
- 全身拉伸瑜伽或散步
- 或完全休息
B站跟练视频推荐
🍽️ 三餐搭配
早餐
同周一，可将面包换成玉米
午餐
同周二，可加入一份蒸豆腐
晚餐
同周二，可替换为冬瓜汤
核心提示
给身体修复时间，为后续训练储能。休息也是训练的一部分。
周四 上肢+核心
🏃 运动计划
热身 (5分钟)
- 重点活动上肢：手臂画圈、肩部环绕
针对性训练 (25分钟)
- 钻石俯卧撑（或跪姿宽距）：3组，每组做到力竭（组间休息60秒）【紧致手臂】
- 俯身YTW伸展：每个字母3组，每组10次（无休息连续做）【改善圆肩驼背】
- 仰卧卷腹：3组，每组15-20次（组间休息30秒）【上腹部】
- 仰卧交替抬腿：3组，每条腿15-20次（组间休息30秒）【下腹部】
- 俄罗斯转体：3组，每侧15次（组间休息30秒）【侧腹】
拉伸 (5分钟)
- 重点拉伸胸、肩、腹、背
B站跟练视频推荐
🍽️ 三餐搭配
早餐
无糖酸奶150g + 坚果一小把（10g） + 燕麦片30g
午餐
鸡胸肉沙拉（含生菜、圣女果、黄瓜） + 油醋汁调味
晚餐
清蒸鲈鱼（半条） + 凉拌秋葵 + 不摄入主食
核心提示
专注手臂和腰腹的发力感，慢起慢落。动作质量第一。
周五 HIIT燃脂
🏃 运动计划
热身 (5分钟)
- 充分热身，避免受伤
HIIT循环 (20分钟)
- 每个动作做40秒，休息20秒
- 完成一组循环后休息1分钟，共3-4组循环
- 动作组合：开合跳、深蹲跳、爬山式、原地高抬腿
拉伸 (5分钟)
B站跟练视频推荐
🍽️ 三餐搭配
早餐
全麦馒头一小个 + 鸡蛋1个 + 黑咖啡一杯（可选）
午餐
瘦肉炒蔬菜（如西芹炒牛肉） + 少量米饭（半拳头）
晚餐
番茄菌菇汤 + 煎鸡胸肉（不加油或少油）
核心提示
HIIT强度大，但后燃效应强，量力而行。保持水分补充。
周六 趣味有氧
🏃 运动计划
趣味有氧或核心强化 (30-40分钟)
- 选择B站喜欢的舞蹈有氧或趣味减脂操视频跟练
- 或进行核心强化训练，动作同周五，增加组数
B站跟练视频推荐
🍽️ 三餐搭配
早餐
可稍丰富，如蔬菜鸡蛋饼
午餐
外出就餐可选日式定食（烤鱼+米饭+味增汤）
晚餐
正常健康晚餐，享受烹饪乐趣
核心提示
保持运动趣味性，有助于长期坚持。周末可适当放松饮食。
周日 完全休息
🏃 运动计划
全天休息
- 可进行轻柔的拉伸或散步
🍽️ 三餐搭配
早餐
享受一顿丰盛但健康的早餐
午餐
正常健康午餐
晚餐
正常健康晚餐，七分饱
核心提示
充分休息，准备迎接新一周。身体恢复同样重要。

居家减脂塑形周计划 · 适合158cm/50.5kg女性

建议每周测量手臂、腰腹围度，关注身体变化
