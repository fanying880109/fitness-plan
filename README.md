# fitness-plan

主理人的居家健身管理页。适合直接通过 GitHub Pages 在 iPhone 上添加到主屏幕使用。

当前功能：

- 可爱像素风 UI
- 今日训练打卡
- 月度日历打卡看板
- Apple Watch 运动卡路里手动记录
- 周训练计划
- 每日三餐搭配
- B 站跟练搜索入口

## 在线访问

启用 GitHub Pages 后，访问地址通常是：

https://fanying880109.github.io/fitness-plan/

## 发布设置

在 GitHub 仓库页面依次进入：

Settings -> Pages -> Build and deployment

选择：

- Source: Deploy from a branch
- Branch: main
- Folder: / (root)

保存后等待 1-2 分钟，GitHub 会生成访问链接。
